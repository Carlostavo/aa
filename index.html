Configuración Segura

Las credenciales de Supabase están protegidas mediante variables de entorno y no son visibles en el código fuente.

Para configurar este sistema en producción, asegúrate de:

  1. Configurar las variables de entorno en tu servidor
  2. Implementar políticas CORS adecuadas en Supabase
  3. Utilizar HTTPS para todas las conexiones
